```sh-session

$ alephium-cli COMMAND

$ alephium-cli (--version)
@h0ngcha0/alephium-cli/0.0.2 linux-x64 node-v14.20.0
$ alephium-cli --help [COMMAND]
USAGE
  $ alephium-cli COMMAND
...
```
